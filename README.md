**Hi everyone,**  
I created a Player Record Management system for my clan, and I thought—why not share it with all of you? Maybe you’ll find the idea interesting, or even find the `.exe` file useful.  

**How to install:**  
1. Download the `.zip` file.
2. Once downloaded, unzip it. Inside, you’ll find three empty `.csv` files and the `.exe` program, which you can run by simply clicking on it.

That’s it! Every record you add, delete, or edit will automatically be updated in the `.csv` files as well.  

**Features:**
1. Search across three tables: *Blacklisted*, *Removed*, and *Clan*.  
2. Easily move player records between tables.  
3. Undo any move you make.  
4. Refresh tables for up-to-date data.  
5. Quickly add records to any table.  
6. View records by individual tables or view all at once.
